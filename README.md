**student-database-management-system**
A streamlined CRUD application for managing university student records, enrollments, and academic statuses in a centralized relational database.

### 01 — INSTALL

Bash
> git clone https://github.com/ciellamher/student-database-management-system.git
> cd student-database-management-system
> npm install

---

### 02 — USE

> npm start

Access the web-based administrative dashboard to insert, update, or query student data.

---

### 03 — WHAT'S INSIDE

- `server.js` — Express API endpoints handling CRUD operations.
- `db_config.js` — Database connection pooling and schema definitions.
- `public/` — The frontend administrative interface files.
